Необходимо сделать небольшое приложение.
Это будет страница поиска репозиториев на github (у github для этого есть открытый API).

По сути мы должны увидеть простой интерфейс:
1. Строка ввода поискового запроса
2. Под строкой карточки найденых репозиториев или надпись, что ничего не найдено
3. Рядом со строкой поиска кнопка, которая фильтрует все найденые репозитории по какому-то параметру (на своё усмотрение, по какому параметру)
4. Рядом со строкой поиска кнопка, которая сортирует все найденые репозитории по какому-то параметру (на своё усмотрение, по какому параметру)

Внешний вид на ваше усмотрение и чувство прекрасного. Вёрстка тоже будет оцениваться, но при этом может быть максимально простой.

Реализовать на чистом JS или если есть навык или желание быстро разобраться на фреймворке Angular.
